## Local Development

```bash
# Clone & init submodules
git clone https://github.com/scinttt/my-blog.git
cd my-blog
git submodule update --init --recursive

# Run local server
hugo server -D
```

## Create a Post

```bash
hugo new posts/my-post.md
```
